# bittergit-marketing

Marketing site for [bittergit.com](https://bittergit.com) — agent-native source control.

## Stack

- Nuxt 4 (static)
- Tailwind 4 (CSS-first config)
- Deployed to Radicchio

## Develop

```
npm install
npm run dev
```

## Deploy

```
Scripts/deploy
```

Requires `RADICCHIO_API_TOKEN`.
